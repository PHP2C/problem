## 虚拟列



### 给二维的数据添加虚拟列
```shell
alter table `table_name` add `virtral_column` varchar(255) generated always as(`column`->"$[*].conclusion_id") stored;
```

### 给一维的数据添加虚拟列
```shell
alter table `table_name` add `virtral_column` varchar(255) generated always as(`column`->"$.conclusion_id") stored;
```
